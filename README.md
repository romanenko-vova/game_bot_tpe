На этом уроке мы научились с Вами инициализировать бота и добавлять обработчики команд.

1. Создать у себя папке C://programs/game_bot
2. Создать виртуальное окружение: Ctrl + Shift + P -> Python: Create Environment -> New Environment -> venv
3. Установить библиотеки: pip install python-telegram-bot
4. Создать файл .env и добавить токен бота: TOKEN=YOUR_TOKEN
5. Создать файл main.py и добавить код урока.

6. В боте botfather зарегистрировать своего бота и получить токен.

7. Зарегистрироваться на github или войти по инструкции из видео добавить своего бота на github.
