На этом уроке мы научились с Вами пользоваться ConversationHandler для создания диалога с ботом.

1. Воспроизведите у себя весь код из урока.

2. Допишите функцию talk, чтобы она обрабатывала 5 рандомных фраз.

3. Сделайте новую команду /game, которая будет запускать игру "Угадай число".
(функция game_start) 

4. Функция game_start должна переводить вас на состояние GAME.

5. В состоянии GAME должна быть функция game, которая запускается, когда человек пишет любой текст. Если этот текст — число. То она должна писать в телеграм слово "число", если нет, то она пишет "не число"
